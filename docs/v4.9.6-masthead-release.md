# v4.9.6 — Fully Customizable Masthead System

**Release date:** March 10, 2026  
**Tests:** 830/830 passing | **TS errors:** 0  
**Breaking changes:** None

---

## Summary

The `brand_masthead_style` preset concept has been removed and replaced with **12 individual masthead controls**. Every visual aspect of the site header is now independently configurable. Two deployments can use the same layout but look completely different because of different colors, fonts, and spacing.

---

## New Settings (all in `DEFAULT_SETTINGS`, `category: "branding"`)

| Key | Type | Default | Description |
|---|---|---|---|
| `masthead_bg_color` | color | `""` | Background color. Falls back to `brand_color_primary` if empty. |
| `masthead_text_color` | color | `#ffffff` | Text color for site name and tagline. |
| `masthead_font_family` | dropdown | `serif` | `serif`, `sans-serif`, or any Google Font name (e.g., `Playfair Display`). |
| `masthead_font_size` | dropdown | `large` | `small`, `medium`, `large`, `extra-large`. |
| `masthead_layout` | dropdown | `center` | `center`, `left`, or `logo-only` (logo image, no text). |
| `masthead_show_tagline` | toggle | `true` | Show or hide the tagline below the site name. |
| `masthead_show_date` | toggle | `true` | Show or hide the date bar above the masthead. |
| `masthead_date_bg_color` | color | `""` | Date bar background. Falls back to `masthead_bg_color` if empty. |
| `masthead_logo_height` | number | `60` | Logo height in pixels. Width scales proportionally. |
| `masthead_padding` | dropdown | `medium` | `compact`, `medium`, or `spacious` vertical spacing. |
| `masthead_border_bottom` | toggle | `false` | Show a border line below the masthead. |
| `masthead_border_color` | color | `""` | Border color when `masthead_border_bottom` is enabled. |

---

## Rendering Logic

**Logo-only layout** (`masthead_layout = "logo-only"` + `brand_logo_url` set):  
Shows ONLY the logo image at `masthead_logo_height` px. No text. Centered or left-aligned based on layout.

**Logo + text layout** (`masthead_layout = "center"` or `"left"` + `brand_logo_url` set):  
Shows the logo alongside the site name and optional tagline.

**Text-only layout** (no `brand_logo_url`):  
Shows the site name in the chosen font family, size, and color. Tagline below if `masthead_show_tagline = true`.

**Date bar:**  
Renders above the masthead when `masthead_show_date = true`. Uses `masthead_date_bg_color` if set, otherwise falls back to `masthead_bg_color`, then `brand_color_primary`.

**Scroll behavior:**  
Masthead shrinks on scroll (font size reduces, padding compresses) and expands when the user returns to the top. This is handled by a `scrolled` state in `Navbar.tsx`.

## Wizard Screen 1 — Live Preview

A **Masthead Preview** section has been added to the Brand & Identity screen in the Setup Wizard. The preview renders in real-time as the operator changes settings — they see exactly what visitors will see before saving.

The preview includes:
- Date bar (if `masthead_show_date = true`)
- Masthead body with correct background color, padding, and alignment
- Logo (if `brand_logo_url` is set and layout is `logo-only`)
- Site name in the chosen font family, size, and color
- Tagline (if `masthead_show_tagline = true` and `brand_tagline` is set)
- Bottom border (if `masthead_border_bottom = true`)

## White-Label Compatibility

All 12 settings are in `DEFAULT_SETTINGS` with sensible defaults. A fresh deployment gets a functional masthead immediately. Each deployment customizes independently via Admin → Settings → Branding or the Setup Wizard. No Hambry-specific values are hardcoded anywhere in the engine.
